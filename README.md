 Employee Management System

This is a simple Employee Management System developed using Core Java as part of my Java Developer Internship at Maincrafts Technology.

The application is menu-driven and runs in the console. It helps manage employee records by allowing users to add, view, search, and update employee information.

 Features

* Add a new employee
* View all employees
* Search employee by ID
* Update employee department
* Exit the application

 Technologies Used

* Java
* OOP (Object-Oriented Programming)
* ArrayList
* Scanner Class

 What I Learned

While building this project, I gained practical experience with:

* Classes and Objects
* Constructors
* ArrayList Collections
* User Input Handling
* Loops and Conditional Statements
* Method Creation
* Building a Menu-Driven Application

 Project Structure

 Employee Class

 Task Class
